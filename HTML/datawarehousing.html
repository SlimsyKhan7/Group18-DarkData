Data Warehousing

Data Warehousing is an extremely useful modern data management system. It creates an effective way to organize and analyze data, helping businesses store substantial amounts of data.

Pros

Efficient

Data Warehousing stores multiple sets of data into one central data store. It can hold large amounts of data, far more than traditional databases.

Analytical

Data warehouses excel at analyzing data, offering numerous use cases such as: AI and machine learning, Business intelligence, Data integration and Industry-specific applications

Environmentally Friendly

Modern cloud data warehouses reduce the need for companies to maintain their own data centers, thereby minimizing environmental impact.

Types

Cloud Data Warehouse

Cloud data warehouses are designed to operate on the web. Offered as a service, they provide large companies with efficient data storage solutions, managed by an external provider. This approach allows companies to avoid the cost and effort of creating less effective in-house solutions.

On-premises Data Warehouse

On-premises data warehouses require the purchase of a license and the installation of a data warehouse at the organization's location. This more costly option is suitable for entities like governments or banks that require higher data regulation and control.

Architecture

Bottom Tier

The bottom tier stores data from multiple sources in a single data warehouse server. Data is processed using one of two methods: ETL (extract, transform, load) or ELT (extract, load, transform), preparing it for further analysis.

Middle Tier

This tier employs an analytics engine to process large volumes of structured and unstructured data, often using OLAP (online analytical processing). OLAP has three types:

  • ROLAP (Relational OLAP): Allows for examining relational databases from multiple perspectives.
  • MOLAP (Multidimensional OLAP): Uses array-based storage for multidimensional data analysis.
  • HOLAP (Hybrid OLAP): Combines features of ROLAP and MOLAP for flexible analysis.

Top Tier

This tier features a user-friendly interface or reporting tool, enabling users to analyze company data. Self-service business intelligence helps users generate reports, identify opportunities, and locate process inefficiencies.
